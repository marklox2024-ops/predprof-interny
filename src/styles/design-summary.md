# Дизайн ОлимпИУМ

## Цветовая палитра

### Основные цвета
- Фон: `from-white via-blue-50 to-sky-50`
- Карточки: `bg-white` с `border-gray-100`
- Текст: `text-gray-800` (заголовки), `text-gray-600` (описания)

### Акцентные цвета
- Основной акцент: `blue-400` (голубой)
- Вторичный акцент: `sky-400` (светло-голубой)
- Градиенты: `from-blue-400 to-sky-400`

### Тени
- Основные: `shadow-md`
- Hover: `shadow-xl`
- Цветные: `shadow-blue-200`

## Компоненты

### Кнопки
- Основные: `bg-gradient-to-r from-blue-400 to-sky-400 text-white`
- Вторичные: `bg-white border border-gray-200 text-gray-700`

### Карточки
- Фон: `bg-white`
- Граница: `border border-gray-100`
- Radius: `rounded-xl` или `rounded-3xl`

### Header
- Фон: `bg-white/90 backdrop-blur-sm`
- Граница: `border-b border-gray-100`
